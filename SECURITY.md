# Security Policy

## ⚠️ Important Notice

PhishShield is an **educational tool** built for cybersecurity awareness training. It contains simulated phishing examples for learning purposes only.

---

## Supported Versions

| Version | Supported |
|---------|-----------|
| 2.x (current) | ✅ Yes |
| 1.x | ❌ No |

---

## Reporting a Vulnerability

If you discover a security vulnerability in PhishShield itself (e.g., XSS in the app, localStorage injection, etc.), please **do not** open a public GitHub issue.

Instead:
1. Email: `security@your-domain.com`
2. Include: description, steps to reproduce, impact assessment
3. We'll respond within 48 hours and credit you in the fix

---

## Scope

**In scope:**
- Cross-site scripting (XSS) in the app UI
- localStorage data exposure
- API key leakage
- Any vulnerability that could harm real users

**Out of scope:**
- The simulated phishing content itself (it's intentionally educational)
- Third-party services (Anthropic API, Google Fonts)

---

## Responsible Use

This tool is for **defensive education only**. Misusing PhishShield or its content to conduct real phishing attacks is illegal and a violation of this project's license. We actively discourage any malicious use.
